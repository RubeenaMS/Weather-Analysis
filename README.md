# Weather Analysis Project

##  Description
This project is a Python-based data analysis tool that processes weather records from five different countries (Bahrain, UK, Singapore, Canada, and Japan). It performs data cleaning, calculates statistical averages, and generates visual reports to show climate trends and weather condition distributions.

## Project Structure
* **`Visualization/visualization/`**: Contains the raw input file `data.csv`.
* **`report/`**: The destination folder for all generated charts and the summary report.
* **`main.py`**: The primary Python script containing the analysis and plotting logic.
* **`requirements.txt`**: A list of Python libraries needed to run the project.

## Getting Started
### 1. Prerequisites
Ensure you have Python installed. You will also need the following libraries:
* **Pandas**: For data manipulation and cleaning.
* **Matplotlib**: For creating the charts.
